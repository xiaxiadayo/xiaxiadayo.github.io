📋 留言板说明 & 限制
  • 留言需要 GitHub 账号登录（基于 giscus 系统）。
  • 支持 Markdown 语法，包括代码块 ```code```，格式会被保留。
  • 由于评论通过 iframe 嵌入，无法为每条留言单独添加复制按钮。你可以在 GitHub Discussions 页面中直接复制每条留言内容。
  • 留言的分页、排列由 giscus/GitHub Discussions 控制，外层折叠/展开功能已提供。
  • "管理模式"密码仅为前端 UI 门槛，实际管理权限由 GitHub 仓库权限控制。
