    کدهای کوتاه

    کدهای کوتاه Hugo قطعه‌های ساده‌ای در داخل پرونده‌های محتوای شما هستند که قالب‌های داخلی یا سفارشی را فراخوانی می‌کنند.

    هگزترا مجموعه‌ای از کدهای کوتاه زیبا را برای بهبود محتوای شما ارائه می‌کند.

    FileTree آیکون جزئیات زبانه‌ها فراخوانی کارت‌ها مراحل
